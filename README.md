# 🛒 E-commerce Superstore Power BI Dashboard  

## 📌 Project Overview  
This project showcases an interactive Power BI dashboard built on the **Superstore dataset**. The analysis provides key business insights into:  

- Best-selling products  
- Sales & profit trends  
- High-revenue categories  
- Customer & regional performance  
- Loss-making products  

The dashboard is designed to help stakeholders make **data-driven decisions** with ease and clarity.

## 🎯 Key Features  

### 3-Page Interactive Dashboard  
- **Page 1 – Sales Overview**: KPIs, Monthly Sales Trends, Category & Regional Performance.  
- **Page 2 – Products & Customers**: Top Products, Top Customers, Sub-Category Analysis, Contribution %.  
- **Page 3 – Profit Insights**: Profit by Category/Region, Scatter Plot (Sales vs Profit), Loss-Making Products Table.  

### Advanced Analytics  
- DAX measures for KPIs, Profit Margin %, YoY Growth, Cumulative Sales.  
- Pareto analysis (80/20 rule).  
- Conditional formatting to highlight losses.  

### User Experience Enhancements  
- Navigation arrows for seamless page movement.  
- Slicers for dynamic filtering (Year, Region, Category).  
- Drillthroughs & tooltips for deep dives.

## 📊 Business Insights  
- **Technology** drives the highest revenue, followed by **Furniture** & **Office Supplies**.  
- **Phones & Chairs** are among the top-selling products.  
- Several **Furniture sub-categories generate losses** despite high sales.  
- **Sales peak during year-end months (Nov–Dec).**  
- A small group of customers (**≈20%**) generates most of the revenue → confirming the **Pareto principle**.
